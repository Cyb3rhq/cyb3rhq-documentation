Cyb3rhq files backup
==================

In this section you can find instructions on how to create and restore a backup of your Cyb3rhq installation.

To do this backup, you copy key files to a folder preserving file permissions, ownership, and path. Later, you can move this folder contents back to the corresponding location to restore your Cyb3rhq data, certificates, and configurations. Backing up Cyb3rhq files is useful in cases such as moving your Cyb3rhq installation to another system.
