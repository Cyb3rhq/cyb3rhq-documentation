Additional security options
===========================

You can implement additional security measures in the enrollment process to authenticate the endpoint to the Cyb3rhq manager and vice versa. These security options are only available when enrolling Cyb3rhq agents via the :doc:`agent configuration method <../enrollment-methods/via-agent-configuration/index>`.
