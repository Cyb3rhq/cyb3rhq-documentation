Enroll agents
=============

To start using Cyb3rhq, you need to install a Cyb3rhq agent on your endpoint and enroll it in your environment.

Follow these steps to enroll an agent:

#. Log into the Cyb3rhq WUI.

#. Click the upper-left **Cyb3rhq** icon and then **Agents**.

#. Click **Deploy new agent**.

#. Follow the steps described on the **Deploy a new agent** page.

  .. note::

    Cyb3rhq agents use TCP to communicate with your environment.

If you have any questions about the Cyb3rhq Cloud, see the :doc:`Cloud service FAQ </cloud-service/getting-started/starting-faq>`.
