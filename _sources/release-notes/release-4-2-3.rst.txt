4.2.3 Release notes - 6 October 2021
====================================

This section lists the changes in version 4.2.3. Every update of the Cyb3rhq solution is cumulative and includes all enhancements and fixes from previous releases.


What's new
----------

This release includes new features or enhancements.

Cyb3rhq Kibana plugin
^^^^^^^^^^^^^^^^^^^

- Cyb3rhq Kibana plugin is now compatible with Cyb3rhq 4.2.3.

Cyb3rhq Splunk app
^^^^^^^^^^^^^^^^

- Cyb3rhq Splunk app is now compatible with Cyb3rhq 4.2.3.

Resolved issues
---------------

This release resolves known issues.
