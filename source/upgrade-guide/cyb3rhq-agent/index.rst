Cyb3rhq agent
===========

The following sections include instructions to upgrade the Cyb3rhq agent to the latest available version. It is possible to upgrade the Cyb3rhq agents either remotely from the Cyb3rhq manager or locally. 

Upgrading the Cyb3rhq agents remotely is possible by using the ``agent_upgrade`` tool and the Cyb3rhq API. More information about the process can be found in the :doc:`Remote agent upgrade </user-manual/agent/agent-management/remote-upgrading/upgrading-agent>` section.

To perform the upgrade locally, select your operating system and follow the instructions.
